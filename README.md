# Análisis Exploratorio de Solicitudes de Crédito con Python y DuckDB

## Descripción

Este proyecto corresponde al trabajo final del módulo **Friendly SQL & Python para Analytics**. Se realizó un análisis exploratorio de un conjunto de datos sobre solicitudes de préstamo utilizando Python, Pandas, Plotly y DuckDB.

## Objetivo

Analizar las características de los solicitantes de crédito mediante estadísticas descriptivas, visualizaciones y consultas SQL para identificar patrones relevantes dentro del conjunto de datos.

## Dataset utilizado

**Nombre:** Loan Approval Prediction Dataset

**Fuente:** https://www.kaggle.com/datasets/maramsa/loan-database

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Plotly
- DuckDB
- Google Colab

## Contenido del proyecto

- Carga y exploración del dataset.
- Revisión de la calidad de los datos.
- Estadística descriptiva.
- Visualizaciones interactivas con Plotly.
- Consultas SQL utilizando DuckDB.
- Hallazgos y conclusiones.

## Cómo ejecutar el notebook

1. Descarga el archivo `Trabajo_Final_SQL_Python_Diego.ipynb` y el archivo `Copy of loan.xlsx` de este repositorio.
2. Abre el notebook en [Google Colab](https://colab.research.google.com/) (Archivo → Subir notebook) o en Jupyter Notebook/JupyterLab de forma local.
3. Ejecuta las celdas en orden. Al llegar a la celda de carga de datos, sube el archivo `Copy of loan.xlsx` cuando se solicite.
4. Todas las celdas están listas para ejecutarse sin necesidad de configuración adicional.
